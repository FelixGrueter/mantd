# MANTD — Make All Nations To Disciples

Hauptwebsite der gemeinnuetzigen Organisation MANTD. Mehrsprachig (DE/EN/IT).

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- GSAP (Animationen via CDN)
- Google Fonts: Bebas Neue, DM Sans, Caveat
- Eigenes i18n-System (translations.js)

## Projektstruktur

```
index.html              Hauptseite
style.css               Stylesheet
translations.js         Uebersetzungen (DE/EN/IT)
assets/images/          Bilder (Logos, Team, Events, Werke)
```

## Lokal starten

```bash
npx serve .
```
